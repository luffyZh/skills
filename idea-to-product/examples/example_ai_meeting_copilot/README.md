# example_ai_meeting_copilot

## 示例定位

这是一个基于 `idea-to-product` 方法论生成的完整示例项目。  
示例主题是：**AI 会议助手**

目标是帮助团队把“会议记录、任务提炼、会后跟进”从手工流程升级为 AI 驱动的协作流程。
