# example_unmanned_patrol_system

## 示例定位

这是一个更贴近真实产业项目的示例，主题是：

**无人装备 AI 巡航系统**

核心思路不是开发无人机或机器狗本体，而是基于成熟本体设备，开发：

- 感知识别算法
- 多设备协同调度
- 场景化巡航应用
- 上层管理与处置系统

## 适合验证的能力

这个示例特别适合验证 `idea-to-product` 是否真的适用于：

- 解决方案型产品
- 多设备协同系统
- 本体设备 + 上层应用的产品模式
- 强场景、强行业、强交付导向的项目
