# PRD

## 文档定位

本文件用于定义产品目标、用户、场景、功能与指标。  
它服务于产品实现、MVP 收敛与跨角色协作。

说明：

- `PRD.md` 应包含精简版“机会摘要 / 市场与竞品摘要”
- 完整的市场规模、竞品格局、替代方案与切入机会，优先来自 `01_opportunity_brief.md`
- 不建议把全量市场研究正文堆入 PRD
