# docs templates

本目录用于放置执行型和表达型核心文档模板。

包含：

- `PRD.template.md`
- `AGENTS.template.md`
- `DESIGN.template.md`
- `MVP_PLAN.template.md`
- `PITCH_DECK.template.md`
- `SOLUTION_BRIEF.template.md`
- `SCENARIO_PLAYBOOK.template.md`

适用时机：

- 当方向已初步收敛
- 当需要开始生成核心交付物
- 当需要进入 Demo 规划、系统设计或汇报准备

补充说明：

- 如果项目属于“成熟设备本体 + 上层系统”的解决方案型产品，建议同时使用：
  - `SOLUTION_BRIEF.template.md`
  - `SCENARIO_PLAYBOOK.template.md`
